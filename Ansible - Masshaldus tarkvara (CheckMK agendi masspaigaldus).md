# Küsimus 1: Mida teeb host_key_checking = False?
Vastus: See rida keelab SSH-võtmete kontrollimise (Known Hosts).

● Miks see kasulik on? Tavaliselt küsib SSH esmakordsel ühendumisel: "Are you sure you want
to continue connecting (yes/no)?".
Automaatika puhul (nagu Ansible) jääks protsess siinkohal
seisma, oodates kasutaja sisendit.● Laboris: Kuna loome ja kustutame virtuaalmasinaid tihti, muutuvad nende identiteedid. See
seade lubab Ansiblel kohe tööle hakata ilma käsitsi kinnitamata.
