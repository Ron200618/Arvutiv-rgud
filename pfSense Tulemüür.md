# Seadista talle 1 WAN ning 3 LAN võrgu liidest.
Tuleviku mõttes soovitan teha algseadistusest snapshoti.
Luba WAN liideselt admin paneelile juurdepääs

Seadista 2 LAN võrku:

- LAN 2: 100.100.100.1/24

- LAN 3: 200.200.200.1/24

# Ülesanne 1:

 

LAN 2 võrk: Luba kõik algatatud ja internetti suunduvad HTTP ja HTTPS paketid

 

LAN3 võrk: Keela kõik algatatud ja internetti suunduvad HTTP ja HTTPS paketid

 

* Katseta inkognito režiimis, veebilehitseja vahemälu võib segada



# Ülesanne 2:

 SSH TESTIMISEKS ON TEIL VAJA LUBADA SYSTEM -> ADVANCED ALT SECRURE SHELL

LAN 2 võrk: Luba ruuteri pihta minevad SSH päringud

 

LAN 3 võrk: Keela ruuteri pihta minevad SSH päringud


# Ülesanne 3:
